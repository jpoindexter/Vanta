# Executive Function Skills

Seven diagnosis-free skills for reducing friction between an intended outcome and the next executable action.

| Skill | Use it for |
| --- | --- |
| `executive-function-router` | Match the current task state to the least intrusive support. |
| `functional-minimums` | Preserve a safe, useful outcome at low capacity. |
| `task-decomposition` | Turn an ambiguous outcome into executable proof steps. |
| `working-memory-externalizer` | Keep active state out of the user's head. |
| `interest-based-initiation` | Create a truthful bridge into a hard-to-start task. |
| `predictable-low-load-communication` | Reduce language, sensory, and uncertainty load. |
| `time-and-transition-support` | Externalize estimates, checkpoints, stops, and handoffs. |

## Design Contract

- Supports are available to anyone and never require diagnostic disclosure.
- Current state is explicit, temporary, user-controlled, and overridable.
- The system modifies the task or environment before demanding more effort.
- Safety, consent, data integrity, and verification remain intact at every capacity level.
- The assistant continues useful work instead of turning support into a coaching ritual.

## Sources

The pack synthesizes and paraphrases concepts from:

- Peg Dawson and Richard Guare, the *Smart but Scattered* series
- Judith Kolberg and Kathleen Nadeau, *ADD-Friendly Ways to Organize Your Life*
- KC Davis, *How to Keep House While Drowning*
- Tamara Rosier, *Your Brain's Not Broken Workbook*
- Megan Anna Neff, *Self-Care for Autistic People*
- Devon Price, *Unmasking Autism*
- University of Oxford, *NESTL Toolkit*

No source text or personal profile is bundled. These are task-support patterns, not medical advice or diagnostic instruments.
