# AGENTS.md — Vanta website

Static marketing-site concept for Vanta. Keep it original: inspired by strong agent landing pages, but do not clone Hermes/Nous visual assets, copy, layout, or branding.

## Files

- `index.html` — self-contained static site with embedded CSS/JS.
- `README.md` — usage notes.

## Verification

Run from repo root:

```bash
python3 -m http.server 4173 --directory vanta-website
```

Then open `http://127.0.0.1:4173`.
