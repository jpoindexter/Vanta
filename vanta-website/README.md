# Vanta Website

A standalone static marketing-site concept for Vanta.

Open directly:

```bash
open vanta-website/index.html
```

Or serve locally:

```bash
python3 -m http.server 4173 --directory vanta-website
# http://127.0.0.1:4173
```

The site is self-contained: no build step, no external assets, no external fonts.
