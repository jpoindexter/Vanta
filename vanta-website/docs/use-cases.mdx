# Use cases

What you can build with Vanta — grounded in its actual capabilities, each linking to the feature that powers it. These are illustrative use cases, not community testimonials; every one maps to shipped functionality.

:::info Executed community-story proof
Vanta indexes all 262 Hermes story records without copying quote bodies and has 30 executable scenarios — two in each of the 15 categories. Five scenarios have explicit reviewed passes: **General / capability start**, **Meta & ecosystem / cross-platform install**, **Dev workflow / converse before act**, **Cost optimization / context reduction**, and **Enterprise / Kubernetes briefing**. These runs exposed and fixed a choice-wall side effect, relative glob-root resolution, missing live context-cost measurement, criterion-free workflow validation, and false repair-loop detection. The remaining 25 scenarios stay unproved until their outcome verifiers run.
:::
