# CLAUDE.md — Vanta website

This folder is a standalone static Vanta marketing website prototype: a single self-contained `index.html` (~510 lines, embedded CSS + JS, no build step, no external assets/fonts). Sections: hero, capabilities, boundary, operator, install (copy-to-clipboard install command). `README.md` has run notes; `AGENTS.md` has the file/verify map.

Design posture:
- Original Vanta identity, not a Hermes clone.
- Dark operator-console base with warm amber signal accents.
- No hyper-realistic human imagery.
- Clear, literal, trust-forward copy.
- Keep external dependencies at zero.

Do not add build tooling unless requested.
