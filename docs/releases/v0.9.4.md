# Vanta Desktop v0.9.4

Vanta Desktop is the native macOS surface for the Vanta agent. The signed app ships with the local TypeScript runtime and launches the separately gated Rust kernel so chat, tools, memory, approvals, and remote channels share one execution boundary.

The release also publishes checksum-paired Vanta kernel binaries for Apple Silicon macOS, Intel macOS, ARM64 Linux, x86_64 Linux, Android/Termux, and x86_64 Windows.

## Inside the app

- **Work:** run the agent, steer active tasks, inspect tool activity, and keep model and approval scope visible.
- **Connect:** configure model providers, skills, MCP servers, messaging channels, and Google services with readiness states.
- **Models:** search the live provider catalog and choose a default or task-specific model.
- **Approvals:** review the proposed action, target, diff, and risk before allowing consequential work.
- **Outputs:** inspect files, previews, receipts, and other artifacts produced by a run.
