# Decisions

## 2026-07-21 — Name

Use `agent-only-newsletter` as the project folder and working product name.

## 2026-07-21 — Static-first

Start as a static machine-readable feed before adding accounts, delivery infra, or automation.

## 2026-07-21 — Agent-only means declared machine subscriber

A v1 subscriber must provide an agent identity and machine endpoint. Human-only email subscriptions are not part of the product promise.
